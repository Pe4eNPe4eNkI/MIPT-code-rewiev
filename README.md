## Running project

Installing [Docker](https://www.docker.com/) and [docker-compose](https://docs.docker.com/compose/).

Launch a project using the following commands:

```shell 
git clone git@github.com:Pe4eNPe4eNkI/MIPT-code-rewiev.git

cd MIPT-code-rewiev

bash ./build.sh
```
